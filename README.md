Data Engineering Enterprise HR uses the Databricks Platform and the Medallion Architecture to ingest data, clean the data ensuring that there is no invalid values and provide clean data to serve business reports. It includes Databricks Job Pipelines that run the data workflows bringing them along the Bronze, Silver and Gold Workbooks, after recieving data stored in Microsoft Azure.

## 🏗️ Built Using
**Template:** Databricks <br>
**Languages:** Pyspark, SQL <br>
**Other Technologies Used:**  Microsoft Azure <br>

## 🚀 Installation
**Prerequisites**
Before running the application, ensure you have the following:
- [Databricks](https://www.databricks.com/) 
- [Microsoft Azure](https://azure.microsoft.com) (install globally using `npm install -g @angular/cli`)

1. Log into Databricks and enter the Databricks Workspace:
<img width="416" height="139" alt="image" src="https://github.com/user-attachments/assets/a6ab2022-80a7-44c0-a08e-5236f542117a" />

2. In the Repos Folder, select "Create Git Folder"
<img width="536" height="299" alt="image" src="https://github.com/user-attachments/assets/64c655a2-91c8-4bb5-a084-d4ab8acea4aa" />

3. Name the Git Folder as "Data Engineering Enterprise HR" or any name that you wish
<img width="806" height="122" alt="image" src="https://github.com/user-attachments/assets/93e3b335-11e8-4a1a-a5e5-25b518d9f7ba" />

4. Select the Create Button and select Git Folder
<img width="545" height="138" alt="image" src="https://github.com/user-attachments/assets/04be1344-9b4d-43ce-b216-262366cf806e" />

5. Place the URL of the Git Repository and press ok
<img width="689" height="243" alt="image" src="https://github.com/user-attachments/assets/a73434d9-f6b7-44aa-8ea3-5dd4141900b7" />

6. Do the Github Authentication if required.
7. Now the Reporistory would appear under Repos in Databricks and you can now access the Notebooks of this Repository
